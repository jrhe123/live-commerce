# live-commerce

Live commerce platform

## Type of IO and Netty

- BIO
- NIO
- AIO

#### BIO

- Blocking IO
- "accept" blocked
- "read" blocked

#### NIO

- Non-Blocking IO
- "accept" is not blocked
- "read" is not blocked

#### AIO

- Asynchronous Non-Blocking IO
- "accept" is not blocked
- "read" is not blocked
- event/signal driven (aio_read)
